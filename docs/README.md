# Documentacion

Este directorio agrupa las guias operativas y de integracion del proyecto.

## Estructura

- users/: documentacion de registro, usuarios y pruebas de onboarding.
- payments/: documentacion de pagos e integracion con Wompi.
- data/: guias de seeding y verificacion de datos.
- bot/: documentacion de integracion tecnica del bot con el backend.

## Archivos

### users

- users/INSOMNIA_USER_ONBOARDING_FLOW.md: flujo de prueba en Insomnia para creacion administrativa y activacion de usuarios.
- users/USER_ENDPOINTS_GUIDE.md: diferencia entre registro publico y alta administrativa de usuarios.

### payments

- payments/WOMPI_FRONTEND_GUIDE.md: integracion del frontend con Wompi.
- payments/WOMPI_RAFFLES_GUIDE.md: flujo de pagos de rifas con Wompi.

### data

- data/SEEDING_GUIDE.md: guia para poblar y verificar datos de prueba.

### bot

- bot/BOT_INTEGRATION_GUIDE.md: autenticacion, variables, endpoints y troubleshooting del bot.
